## 🔄 State Management

### User States
1. **Guest** → Landing page with 3 options
2. **Profile Created** → Can browse/request
3. **Team Leader** → Full management
4. **Team Member** → View team, can leave
5. **Voting** → Must vote for leader
6. **No Team** → Redirect to create/browse

### Team States
1. **Recruiting** → Accepting requests
2. **Full** → All roles filled
3. **Voting** → Electing new leader
4. **Closed** → Not recruiting
5. **Disbanded** → No members left

### Navigation Flow
```
Landing Page
    ├── Start Team → Profile → Create Team → Leader Dashboard
    ├── Join Team → Profile → Browse → Request → Member View
    └── Manage Team → Login → Dashboard/Redirect
```
